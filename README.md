# Ressai-agrinho-# 🌾 AgroSustentável: Guardião do Solo

Projeto desenvolvido para o concurso **Agrinho 2026**. O foco desta aplicação é promover a conscientização sobre a saúde do solo como pilar fundamental para um agronegócio forte e sustentável.

## 🚀 Funcionalidades
- **Diagnóstico de Saúde:** Entrada de dados técnicos (pH, Matéria Orgânica).
- **Dicas de Sustentabilidade:** Sugestões de rotação de culturas e plantio direto.
- **Interface Responsiva:** Funciona no celular e no computador.

## 🌱 Por que focar no solo?
Um solo saudável retém mais água, estoca carbono e exige menos fertilizantes químicos, garantindo a produtividade a longo prazo.
